# כלי תכנון משכנתא (Mortgage Planning Tool)

כלי מקצועי לתכנון משכנתא עם שמירה מקומית בלבד.

## תכונות

- הזנת מחיר הבית בשקלים חדשים
- הגדרת תנאי המשכנתא (משך וריבית)
- הזנת 3 המשכורות האחרונות לכל אחד מבני הזוג
- ניהול חסכונות (שם, סכום, תשואה, מצב)
- ניהול הלוואות (שם, סכום, משך, תשלום חודשי)
- ניהול הוצאות צפויות
- הגדרת מס רכישה (דירה ראשונה/שנייה, עד 6 רמות מס)

## התקנה

```bash
npm install
```

## הרצה

```bash
npm run dev
```

## בנייה

```bash
npm run build
```

## טכנולוגיות

- React + TypeScript
- Tailwind CSS
- Vite
- LocalStorage

## הערות

- כל הנתונים נשמרים אוטומטית ב-localStorage
- הממשק בעברית עם תמיכה ב-RTL
- מוכן לשילוב אלגוריתמי חישוב בעתיד
